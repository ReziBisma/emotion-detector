# Emotion Detector App

This project is a web-based AI application that detects emotions from text using IBM Watson NLP.

## Features
- Detects emotions: anger, disgust, fear, joy, sadness
- Returns dominant emotion
- Flask web interface

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run server:
python server.py

3. Open browser:
http://localhost:5000/emotionDetector?textToAnalyze=I am happy
